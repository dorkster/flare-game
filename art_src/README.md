# Art source files

**The source files for the newer "HD" art can be found in the [flare-game-art-src](https://github.com/flareteam/flare-game-art-src) repository.**

The files in this folder remain as a historical reference. Many of the Blender files require an older version of Blender (2.7 or earlier). You can read more about how/why the art was transitioned here: https://github.com/flareteam/flare-game/issues/986
